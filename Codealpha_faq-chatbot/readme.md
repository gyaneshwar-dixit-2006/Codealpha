# 🤖 AI FAQ Chatbot

An intelligent FAQ Chatbot developed as part of the **CodeAlpha Internship Program**. This chatbot provides instant answers to frequently asked questions through an interactive and user-friendly conversational interface, helping users quickly find relevant information without manual assistance.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/faq-chatbot.git
```

2. Navigate to the project folder

```bash
cd faq-chatbot
```

3. Run the project

```bash
python app.py
```

4. Open your browser and start chatting!
